# 渐进性原理及表示符号

使用渐进性原理对于算法的时间复杂度进行分析，反映算法的时间复杂度随着N变化发生变化的情况，衡量了算法的规模。

使用渐进分析的专用记号对于渐进性进行分析：

  • 渐进上界记号O
  • 渐进下界记号Ω
  • 非紧上界记号o
  • 非紧下界记号ω
  • 紧渐进界记号Θ

渐进分析中中的符号类似于比较：

  • f(n)=O(g(n))≈a≤b
  • f(n)=Ω(g(n))≈a≥b
  • f(n)=Θ(g(n))≈a=b
  • f(n)=o(g(n))≈a<b
  • f(n)=ω(g(n))≈a>b

同时渐进分析记号还具有若干性质：

  1. 传递性

  2. 反身性

  3. 对称性

  4. 互对称性

  5. 支持算术运算

    O(f(n)+g(n))=O(max{f(n),g(n)})

    O(f(n))+O(g(n))=O(f(n)+g(n))

    O(f(n))×O(g(n))=O(f(n)×g(n))

    O(cf(n))=O(f(n))

# 递归方程渐进阶的求解

# 代入法

先推测递归方法的显式解，然后使用数学归纳法证明这一推测的正确性。

例： 求证T(N)=2T(⌊N2⌋)+n的渐进阶。

首先，推测T(N)=O(nlogn)， 即存在正的常数C和自然数n0，使得当n≥n0时：

T(N)≤Cnlogn

假设当2k−1no≤n≤2kn0，k>1是，上面的推论成立，那么当2kn0≤n≤2k+1n0时，有：

T(n)=2T(⌊n2⌋)+n≤2C⌊n2⌋log(⌊n2⌋)+n<2Cn2log(n2)+n=Cnlogn−Cn+n=Cnlogn−(c−1)n≤Cnlogn

原假设成立。

# 迭代法

迭代展开递归方程的右端，使之成为一个非递归的合式，然后通过对合式的估计来达到对于方程左端解的估计。

例：求

T(n)={7,n=12T(n2)+5n2,n>1

的渐进阶。

T(n)=2T(n2)+5n2=2(2T(n4)+5(n2))2+5n2=2(2(2T(n8)+5(n4)2)+5(n2))2+5n2=2kT(1)+2k−15(n2k−1)2+⋯+2×5(n2)2+5n2

不难发现：

T(n)=O(n2)

# 套用公式法

针对如下的递推方程

T(n)={c,n=1aT(nb)+cnk,n>1

我们有

T(n)={O(nlogba),a>bkO(nklogbn),a=bkO(nk),a<bk

这个公式还有一般化的情况：

如果递归方程的形式为：

T(n)={c,n=1aT(nb)+f(n),n>1

则针对f(n)进行讨论：

  1. 如果∃ϵ>0, 使得f(n)=O(nlogba−ϵ)，那么我们有

    T(n)=θ(nlogba)

  2. 如果f(n)=θ(nlogba)，那么我们有

    T(n)=θ(nlogba⋅logn)

  3. 如果∃ϵ>0，使得f(n)=Ω(nlogb(a+ϵ))，且当∃c<1时，当n充分大时有af(nb)≤cf(n)，那么我们有

    T(n)=θ(f(n))

# 母函数法

通用的方法总是复杂的。

设a0,a1,⋯,an是任意的数列，那么称下面这个函数为数列的母函数：

f(x)=a0+a1x+⋯+anxn

如果数列是算法的复杂性函数{T(n)}，则其母函数为：

f(x)=T(0)+T(1)x+⋯+T(n)xn

如果能由T(n)也就是的数列的递归方程求出母函数，那么其第n项系数为T(n)。
