算法导论

算法

算法是指解决问题的一种方法或者一个过程。

算法是若干指令的有穷序列：

  • 输入
  • 输出
  • 确定性
  • 有限性

算法的时间复杂度分析

渐进性原理及表示符号

使用渐进性原理对于算法的时间复杂度进行分析，反映算法的时间复杂度随着N变化发生变化的情况，衡量了算法的规模。

使用渐进分析的专用记号对于渐进性进行分析：

  • 渐进上界记号O
  • 渐进下界记号Ω
  • 非紧上界记号o
  • 非紧下界记号ω
  • 紧渐进界记号Θ

渐进分析中中的符号类似于比较：

  • f(n)=O(g(n))≈a≤b
  • f(n)=Ω(g(n))≈a≥b
  • f(n)=Θ(g(n))≈a=b
  • f(n)=o(g(n))≈a<b
  • f(n)=ω(g(n))≈a>b

同时渐进分析记号还具有若干性质：

  1. 传递性

  2. 反身性

  3. 对称性

  4. 互对称性

  5. 支持算术运算

    O(f(n)+g(n))=O(max{f(n),g(n)})

    O(f(n))+O(g(n))=O(f(n)+g(n))

    O(f(n))×O(g(n))=O(f(n)×g(n))

    O(cf(n))=O(f(n))

递归方程渐进阶的求解

代入法

先推测递归方法的显式解，然后使用数学归纳法证明这一推测的正确性。

例： 求证T(N)=2T(⌊2N​⌋)+n的渐进阶。

首先，推测T(N)=O(nlogn)， 即存在正的常数C和自然数n0​，使得当n≥n0​时： T(N)≤Cnlogn 假设当2k−1no​≤n≤2kn0​，k>1是，上面的推论成立，那么当2kn0​≤n≤2k+1n0​时，有： \[ \begin{eqnarray} T(n) &=& 2T(\lfloor \frac{n}{2} \rfloor) + n \ &\le& 2 C \lfloor \frac{n}{2} \rfloor log(\lfloor \frac{n}{2} \rfloor) + n \ &<& 2C\frac{n}{2} log(\frac{n}{2}) + n \ &=& Cnlogn - Cn + n \ &=& Cnlogn - (c-1)n \ &\le& Cnlogn \end{eqnarray} \] 原假设成立。

迭代法

迭代展开递归方程的右端，使之成为一个非递归的合式，然后通过对合式的估计来达到对于方程左端解的估计。

例：求 T(n)={7,n=12T(2n​)+5n2,n>1​ 的渐进阶。 \[ \begin{eqnarray} T(n) &=& 2T(\frac{n}{2}) + 5n^2 \ &=& 2(2T(\frac{n}{4}) + 5(\frac{n}{2}))^2 + 5n^2 \ &=& 2(2(2T(\frac{n}{8}) + 5 (\frac{n}{4}) ^ 2) + 5(\frac{n}{2}))^2 + 5n^2 \ &=& 2^kT(1) + 2^{k-1} 5(\frac{n}{2^{k-1}}) ^ 2 + \cdots + 2 \times 5 (\frac{n}{2})^2 + 5n^2 \end{eqnarray} \] 不难发现： T(n)=O(n2) 迭代法还有一个衍生的方法——递归树法：

如果递归方程的形式为： T(n)={c,n=1aT(bn​)+f(n),n>1​ 则针对f(n)进行讨论：

  1. 如果∃ϵ>0, 使得f(n)=O(nlogb​a−ϵ)，那么我们有 T(n)=θ(nlogb​a)

  2. 如果f(n)=θ(nlogb​a)，那么我们有 T(n)=θ(nlogb​a⋅logn)

  3. 如果∃ϵ>0，使得f(n)=Ω(nlogb​(a+ϵ))，且当∃c<1时，当n充分大时有af(bn​)≤cf(n)，那么我们有 T(n)=θ(f(n))

母函数法

通用的方法总是复杂的。

设a0​,a1​,⋯,an​是任意的数列，那么称下面这个函数为数列的母函数： f(x)=a0​+a1​x+⋯+an​xn 如果数列是算法的复杂性函数{T(n)}，则其母函数为： f(x)=T(0)+T(1)x+⋯+T(n)xn 如果能由T(n)也就是的数列的递归方程求出母函数，那么其第n项系数为T(n)。
